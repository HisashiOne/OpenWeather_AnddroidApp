# OpenWeather AnddroidApp
Android and Widget App with Open Wearher API


  - Get Parameters by GPS Location
  - View a Widget in Home Screen
  
You can view parameters Like:
  
  - City
  - Weather Description
  - Clouds
  - Wind
  - Pressure
  - Humidity
  - Temp Morning
  - Temp Day
  - Temp Night
